# React Text Galaxy

This is a text galaxy animation component based on canvas for React.

![Preview](preview-react-text-galaxy.gif)

## Install

```bash
npm i react-text-galaxy
```

## Import

```typescript
import { TextGalaxy } from 'react-text-galaxy'
```

## Use

```typescript
<TextGalaxy
  text={new Array(2000).fill('.').join('')}
/>
```
